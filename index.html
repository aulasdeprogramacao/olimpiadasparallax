Projeto da Página

Os estudantes do Centro Estadual de Educação Profissional Professora Maria do Rosário Castaldi criaram uma página na internet em HTML utilizando recursos de formatação CSS e linguagem JavaScript para apresentar informações dos esportes disputados nos eventos descritos abaixo:

1. Jogos Olímpicos e Paralímpicos de Verão 2024

2. Jogos Olímpicos e Paralímpicos de Inverno 2026

3. Olimpíadas de Componentes Curriculares (Matemática, Filosofia, Português, Ciência …)

4. Olimpíada Militar

5. Jogos dos Povos Indígenas
